# Основные директории и файлы
- Код backend-сервера: `backend/src/main`
- Код frontend-страниц: `web/src/routes`
- Код frontend-сервера: `web/src/main`
- Код проекта (запуск обоих серверов): `src/main`
- Код сборки проекта (сборка обоих серверов): `build.gradle.kts`
- Код сборки backend-сервера: `backend/build.gradle.kts`
- Код сборки frontend-страниц: `web/svelte.config.js`
- Код сборки frontend-сервера (также исполняет сборку frontend-страниц): `web/build.gradle.kts`
